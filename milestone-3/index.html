Instructions

The game starts when the play button is clicked.

A three minute timer will start.

Enter the number of players.

All players should enter their names.

All players should list down the words they searched from the boggle.

After the timer ended, each player will enter their list of words to an input text provided on the lower part of the game platform.

Done

Word Boggle

is a CMSC 142 project in University of the Philippines Los Baños. It is a boggle game interface with a built-in boggle solver implemented in javascript using an iterative backtracking algorithm and trie data structure.

  • GAME RULES

    The time limit for searching words is three minutes.

    When the timer starts, all players search for the valid words.
    A player can search a word starting from a letter tile to an adjacent letter tile.
    However, a player can only traverse through the adjacent tiles that are not yet visited.

    After three minutes have elapsed, all players should immediately stop writing the words they searched.

    Each player will enter their words to their input box separated by space.

  • SCORING

    In word boggle, you only win points for your words if no other players have found that word.

    Point system will be based on word length.

    Length: 3-41 point Length: 52 points Length: 63 points Length: 7-above5 points
